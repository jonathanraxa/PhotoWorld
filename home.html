PhotoWorld Application

About

Description

Photo World is a web application that allows users to interact with images by way of using geolocation and tag metadata via Flickr concurrently with the Google Maps API. The embedded geolocation in the images will allow the application to track where specifically Google map will find the image. Users shall also show the photo information once they click a marker within the map that will bring up a description of the photo that includes the title, user, date/time, and location of the photo. Another feature included is relational data where a user could find the areas where a particular image tag was used for photos taken.

Current Uses

  • Search for image based on keywords (with dates and other filters)
  • Displays thumbnails on bottom of page
  • Clicking on thumbnail photo will display marker with info window on the map
  • Display multiple photos on map based on locations (without clicking on each photo)
  • Select photos to add to slideshow by way of info window
  • Enter text (comment) for an image, save on browser
  • Display slideshow with comments from info window
  • Save/reload slideshow after closing browser
